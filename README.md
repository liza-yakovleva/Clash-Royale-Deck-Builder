# Clash Royale Deck Builder | React Web Application

**Спеціальність:** Комп'ютерна інженерія  
**Дисципліна:** Розробка інтерфейсів користувача(Лабораторна робота №5)  
**Виконавець:** Яковлева Єлизавета Дмитрівна  
**Семестр:** 8  

---

## Огляд проєкту
Сучасний, адаптивний та повністю інтерактивний веб-застосунок для створення ігрових колод (Deck Builder). Проєкт розроблено як Single Page Application (SPA) з використанням бібліотеки **React**. 

Головна мета застосунку — надати користувачу зручний інтерфейс для конструювання оптимальної 8-карткової колоди з використанням сучасних патернів UI/UX та механіки Drag-and-Drop.

---

## Основні функції (Features)

- **Інтуїтивний Drag-and-Drop:** Плавне перетягування карток між пулом та активною колодою, реалізоване на базі `@dnd-kit/core`. Працює як на ПК, так і на мобільних пристроях.
- **Аналітика в реальному часі:** Автоматичний підрахунок середньої вартості еліксиру (Average Elixir cost) та аналіз балансу колоди (співвідношення військ, заклинань та будівель).
- **Візуалізація даних:** Динамічна побудова кривої еліксиру (Elixir Curve) для візуальної оцінки "важкості" колоди.
- **Розумне фільтрування:** Швидкий пошук карток за типом та сортування за вартістю.
- **Збереження стану (Persistence):** Активна колода автоматично зберігається у локальну пам'ять браузера (`LocalStorage`) за допомогою кастомного React Hook.
- **Преміальний UI/UX:** Сучасний дизайн з використанням ефекту Glassmorphism та CSS Modules для інкапсуляції стилів.

---

## Технологічний стек

- **Фреймворк:** React 18 (на базі збирача Vite для миттєвої компіляції)
- **Стилізація:** Vanilla CSS (CSS Modules)
- **Drag-and-Drop механіка:** `@dnd-kit/core`
- **Архітектура:** Functional Components, Custom Hooks (`useDeckLogic`, `useLocalStorage`), Context API.

---

## Архітектура проєкту (Separation of Concerns)

Проєкт побудовано за модульним принципом для забезпечення масштабованості та чистоти коду:

```text
src/
├── components/
│   ├── CardPool/        # Сітка доступних карток із фільтрами
│   ├── DeckArea/        # Drop-зона для активної колоди (макс. 8 карток)
│   ├── DeckStats/       # Блок аналітики та візуалізації графіка еліксиру
│   ├── DraggableCard/   # Перевикористовуваний UI-компонент картки
│   └── Toast/           # Система кастомних сповіщень
├── data/
│   ├── cards.js         # База даних карток (Mock API)
│   └── constants.js     # Глобальні константи та типи
├── hooks/
│   ├── useDeckLogic.js  # Ізольована бізнес-логіка (DND, ліміти, фільтрація)
│   └── useLocalStorage.js # Логіка синхронізації з пам'яттю браузера
└── App.jsx              # Головний контейнер застосунку
```

## Інструкція із запуску (Локальне середовище)
1. **Встановлення залежностей:**
   ```bash
   npm install
   ```
   
2. **Запуск сервера розробки:**
   ```bash
   npm run dev
   ```

3. **Перегляд:**
  ***Відкрийте в браузері локальне посилання, яке з'явиться у терміналі (зазвичай http://localhost:5173).***
